# E-Shop Database Design (SQL Project)

This project implements the database schema for an e-commerce (e-shop) system as part of a university assignment. It focuses on relational database design, and SQL query development.

## Overview

The project defines the structure of an online shop database using SQL. It includes table creation scripts, relationships between entities, and example queries for managing and retrieving data.

## Database Design

The schema models a typical e-commerce system, including entities such as:

* Customers
* Products
* Orders
* Order items
* Payments / transactions

Relationships are designed using primary and foreign keys to ensure data consistency and relational integrity.

## Features

* Full SQL schema for e-commerce system
* Use of primary and foreign keys
* Example queries for:

  * Inserting data
  * Updating records
  * Retrieving order information

## Technologies Used

* SQL (MySQL) 

## What This Project Demonstrates

* Relational database design principles
* Entity-relationship modeling
* Writing SQL queries with joins
